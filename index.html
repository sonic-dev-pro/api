Instagram Downloader

🟢 نشط

واجهة برمجية لتحميل المحتوى من انستقرام. تدعم: الريلز، المنشورات، القصص، والألبومات. تعيد روابط التحميل المباشرة بأعلى جودة متاحة.

🆓 مجاني ⭐ شائع
رابط الـ API
GET /api/instagram?url={رابط_انستقرام}
CURL
# تحميل ريل من انستقرام curl -X GET "https://api.sonic.dev/api/instagram?url=https://www.instagram.com/reel/C7Xxyz123/"
JavaScript (Fetch)
// تحميل ريل من انستقرام fetch('/api/instagram?url=https://www.instagram.com/reel/C7Xxyz123/') .then(res => res.json()) .then(data => { console.log('أفضل جودة:', data.data.bestQuality.url); console.log('جميع الفيديوهات:', data.data.videos); console.log('الصور:', data.data.images); });
